 Project Overview
An intelligent financial management tool designed to prioritize and manage bills using advanced data structures. The system helps users make data-driven payment decisions while maintaining an emergency reserve.

 Key Features
Smart Prioritization: Uses a Priority Queue to rank obligations based on penalty risks and due dates.

Undo Functionality: Implements a Stack to allow users to reverse recent payment decisions.

Audit Trail: A Doubly Linked List tracks every system action for full transparency.

Robustness: Integrated Try-Catch blocks to handle invalid inputs and date formats.

 Technologies Used
Language: Java (JDK 23).

Data Structures: Priority Queue, Stack, Doubly Linked List.

Concepts: OOP, Encapsulation, Exception Handling.
